변수의 사용

변수 선언 규칙
1.대소문자 구분
2.영문, $, _로 시작한다
3.영문, $, _, 숫자 포함 가능하다
4.키워드, 예약어 사용 불가능하다
변수 범위
1.전역 변수 : window 객체에 포함되는 변수로 다른 함수들에서 공용으로 사용할 수 있다.(값이 유지됨)
2.지역 변수 : 함수나 객체 내부에서 선언 되고 실행이 종료되면 사라진다.
변수 선언, 저장
1.var 키워드를 사용하여 변수 선언
2.변수의 타입은 저장되는 타입에 따라 결정된다.
타입 종류
1.문자(String)
2.숫자(Number)
3.논리(Boolean)
4.null
5.객체
6.함수
